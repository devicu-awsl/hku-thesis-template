A classifier $f(\cdot)$ is a function or mapping which returns for a
feature vector $\mathbf{x}$ the membership function $y$ that belongs
to a given set $S$. The membership function can either be a soft
decision where $y \in [0,1]$, or a hard decision where $S$ consists
of discrete labels of different classes \cite{maronna2006}.
